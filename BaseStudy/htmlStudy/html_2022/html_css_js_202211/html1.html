html基本学习

一、html结构简介

html由几部分组成:
html的标志：DOCTYPE html
html主体：html
html主体里又包含两大模块：
html头：head
html身体：body

1.1 头部

头部包含了几个元素:
title: 定义html文档的标题，会展示在浏览器顶端的网页导航栏里，比如本网站的：“我是标题”
base: 定义了所有链接的默认的相对目标地址，即：本页面出现的其他链接都将在此链接后面添加自己的链接
meta: 表示html文档的描述、关键词、作者、字符集等信息
link: 定义了文档与外部资源的关系，例如本文就引用了同文件夹下的css1.css
style: 样式文件引用地址，也可以直接添加样式来渲染html文档，如本文就在此引用了同文件夹下的css2.css
script: 用于加载脚本文件，如：JavaScript，如本文就在此引用了同文件夹下的js1.js，引用了之后，则该下的脚本则失效了
