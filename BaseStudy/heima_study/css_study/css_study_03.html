一、块元素

块元素包括

h1~h6, p, div, ul, ol, li等，其中div是最典型的块元素

特点

  1. 独占一行
  2. 高度、宽度、外边距、内边距都可以控制
  3. 宽度默认是容器的100%
  4. 是一个容器及盒子，里面可以放行内或 块元素
  5. 文字类的块元素不允许再放块元素了

二、行内元素

行内元素包括

a, strong, b, em, i, del, s, ins, u, span

特点

  1. 可以放在同一行
  2. 不能直接设置高度、宽度
  3. 默认宽度是本身内容的宽度
  4. 只能容纳文本或其他行内元素
  5. a不能再放a，a可以再放块元素

三、行内块元素

包含

img, input, td,

特点

  1. 可以一行放多个（行内）
  2. 默认宽度是内容宽度（行内）
  3. 高度、宽度、外边距、内边距可以控制（块）
